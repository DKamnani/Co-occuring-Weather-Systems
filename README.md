The codes included here are part of the analysis conducted for “Co-Occurring Weather Systems Vary by Atmospheric River Categories Across the United States.” Each notebook contains the code used to generate figures from the data.

Additional Notes:
1. Input datasets are not included in this repository and must be downloaded separately.
2. Figure1_Calc.ipynb – Performs analysis prior to plotting Figure 1.
3. Figure1_Plot.ipynb – Generates Figure 1 using the output from Figure1_Calc.ipynb.
4. Figure2.ipynb – Generates Figure 2 from the main text.
5. Figure3a.ipynb – Generates Figure 3 panels 1(a)–(f).
6. Figure3b.ipynb – Generates Figure 3 panels 2(a)–(f).
7. Plot4.ipynb – Generates Figure 4 in the main text, as well as Figures S1 and S2 in the supplementary section.
8. Supplemental_figure.ipynb – Generates Figure S3 for the supplementary section.
